# CHATMIA

Plataforma de videochat aleatorio en tiempo real tipo Chatroulette / Dirtyroulette.

---

# STACK

## Frontend
- Next.js
- TypeScript
- TailwindCSS
- Framer Motion
- Socket.IO Client
- WebRTC

## Backend
- Node.js
- Express
- Socket.IO
- Redis
- PostgreSQL

## Infraestructura
- Docker
- Nginx
- Coturn TURN Server
